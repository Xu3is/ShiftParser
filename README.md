# Утилита фильтрации содержимого файлов
Консольная утилита для разделения данных из текстовых файлов на целые числа, числа с плавающей точкой и строки, с последующей записью в отдельные текстовые файлы

Программа реализована в рамках тестового задания на курс Java проекта Шифт.

## Используемый стек
Версия Java: 25.0.2

Система сборки: Maven (3.9.11)

Сторонние библиотеки: Не использовались, только стандартна библиотека Java

## Сборка и запуск
1. Склонируйте репозиторий:
```bash
git clone https://github.com/Xu3is/ShiftParser
cd ShiftParser/
```
2. Выполните сборку проекта:
```bash
mvn package
```
3. После сборки исполняемый JAR файл будет создан в папке target, пример запуска:
```bash
java -jar target/Parser-main.jar -s -a -p sample- input/exampleIn1.txt exampleIn2.txt
```

## Примеры входных файлов
В папке `input/` находятся тестовые файлы для удобного тестирования программы.

- `exampleIn1.txt, exampleIn2.txt` — примеры файлов из задания
- `in3.txt, in4.txt, in5.txt, in6.txt, in7.txt, in8.txt` — дополнительные тестовые файлы
